# 💡 9️⃣ Intelligent Prompting Strategy (Critical)

Don’t send plain text like:

> “Generate test for this procedure”

Instead send structured input:

```json
{
  "procedure_name": "Transfer",
  "parameters": [
    {"name": "Amount", "type": "Float", "mode": "IN"},
    {"name": "Balance", "type": "Float", "mode": "IN OUT"}
  ],
  "exceptions": ["Insufficient_Funds"],
  "complexity": 7,
  "conditions": [
    "Amount > 0",
    "Balance >= Amount"
  ]
}
```

AI performs MUCH better with structured data.

**Purpose:** Navigate and inspect procedures/packages.

**Features:**

* Tree view of packages and subprograms

* Streamlit tabs or `st.expander` for each package

# 3️⃣ Test Generation Control Panel

* Streamlit `st.multiselect` for procedures
